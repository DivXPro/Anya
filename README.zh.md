# Anya

[English](README.md) | [中文](README.zh.md)

> 一个放在桌面上的紧凑型语音助手设备 —— 开口说话，Anya 就会回应。

Anya 是一个可以放在你桌面上的免提语音伴侣。按下设备上的通话按钮，你的声音就会被发送到电脑上的 Anya 桌面应用；随后，回答会通过设备的内置扬声器播放出来，同时屏幕会显示一段简短摘要。

你可以让它写代码、查资料、记笔记，或者控制你的工具 —— 只要是你喜欢的本地 AI Agent 能做的事，Anya 都能帮你完成。

---

## Anya 能做什么

- **一键即说语音控制** —— 按住设备上的按钮，说话，然后松开，Anya 会处理剩下的事情。
- **自然语音回复** —— 你的语音在本地电脑上转录为文字，交给 AI Agent 处理，再合成为语音发回设备。
- **可视化反馈** —— 设备屏幕会显示连接状态、识别到的内容，以及回复的简短摘要。
- **多 Agent 支持** —— 在菜单栏或设置窗口中直接切换 Claude Code、Codex、OpenCode、Kimi、Hermes、Pi 等兼容 ACP 协议的 Agent。
- **自动重连** —— 配对一次后，设备会记住你的电脑，下次上线时自动重新连接。
- **无线固件更新** —— 无需插线，直接在桌面应用中为设备更新固件。
- **对话历史** —— 所有对话都保存在你的电脑本地，可在应用中搜索查看。

## 工作原理

```
┌──────────────┐     WiFi / 局域网    ┌──────────────────┐
│  Anya 设备   │  ◄──────────────►  │  Anya 桌面应用   │
│  (M5StickC   │      WebSocket     │  运行在你的 Mac  │
│   S3)        │                    │  或 Windows 上   │
└──────────────┘                    └────────┬─────────┘
                                             │
                              ┌──────────────┼──────────────┐
                              ▼              ▼              ▼
                         ┌────────┐    ┌─────────┐    ┌──────────┐
                         │  STT   │    │  ACP    │    │   TTS    │
                         │ Whisper│    │  agent  │    │ Edge TTS │
                         └────────┘    └─────────┘    └──────────┘
```

1. 按住设备上的通话按钮。
2. 音频通过局域网实时传输到桌面应用。
3. 应用将你的语音转录成文字，发送给当前选中的 Agent，并接收回复。
4. 回复被提炼为屏幕摘要、合成为语音，并在设备上播放。

## 硬件

Anya 围绕 **M5StickC S3**（ESP32-S3）设计，带有麦克风、扬声器、屏幕和按键。你可以使用现成零件自行组装一台。

所需硬件：

- M5StickC S3
- 用于烧录和供电的 USB-C 数据线
- 一台与设备处于同一 WiFi 网络的电脑

## 快速开始

### 1. 安装桌面应用

下载适用于 macOS 或 Windows 的最新版本，或从源码构建：

```bash
./build.sh
```

首次运行时，应用会在你电脑的 `~/.elf/` 目录下创建本地配置和数据。

### 2. 烧录固件

通过 USB-C 将 M5StickC S3 连接到电脑，然后烧录固件：

```bash
cd firmware
pio run --target upload
```

你也可以在桌面应用的 **设置 → 设备固件** 中使用预编译固件镜像进行烧录。

### 3. 为设备连接 WiFi

首次开机时，设备会创建一个名为 **Anya-hotspot** 的热点：

1. 用手机或电脑连接 `Anya-hotspot`。
2. 在浏览器中打开 `http://192.168.4.1/` 进入配网页面。
3. 选择你的家庭 WiFi 网络并输入密码。
4. 设备会重启并加入你的网络。

### 4. 配对设备

打开 Anya 桌面应用：

1. 进入 **设备** 标签页，点击 **扫描**。
2. 从列表中选择你的设备，点击 **连接**。
3. 设备会与你电脑配对，并在下次自动记住这台电脑。

### 5. 选择 Agent 并开始对话

在菜单栏或设置窗口中选择你想使用的 AI Agent，然后按住设备上的通话按钮说话。屏幕会显示回复摘要，设备会朗读完整回答。

## 更新固件

Anya 支持无线固件更新：

1. 先将设备连接到桌面应用。
2. 打开 **设置 → 设备固件**。
3. 如果有可用的新版本，点击 **更新**。
4. 等待进度条完成，设备会自动重启。

---

## 隐私

Anya 采用本地优先的设计：

- 你的语音通过本地 Whisper 模型在电脑上转录。
- 对话历史保存在本地 SQLite 数据库中。
- AI Agent 通过你已控制的本地 ACP 兼容客户端运行。
- 设备与桌面应用仅通过本地网络通信。

## 开发

如果你想自己构建或修改 Anya：

| 模块 | 命令 |
|------|------|
| 桌面应用 + 固件打包 | `./build.sh` |
| 仅桌面应用 | `cd desktop && wails3 build` |
| 运行 Go 测试 | `cd desktop && go test ./... -count=1` |
| 仅固件 | `cd firmware && pio run` |

项目约定请参阅 [`CLAUDE.md`](CLAUDE.md)，硬件相关细节请参阅 [`firmware/README.md`](firmware/README.md)。
